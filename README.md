# spotify-explorer
A web application to explore artists and albums though the Spotify API

## Useful commands
Activate venv  
`.venv\Scripts\activate`

Deactivate venv  
`deactivate`

Run server  
`python manage.py runserver`

Migrate database  
```bash
python manage.py makemigrations
python manage.py migrate
```

Run frontend
`npm run serve`
